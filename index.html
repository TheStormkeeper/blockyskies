BLOCKY SKIES AMIGA 2016

Download the test version of BLOCKY SKIES, a new game for classic Amiga.

This version is designed to run on any PAL 68k Amiga with 512kb of ram.

Please be aware that this code has not been tested!! It is not known what it might do to your precious Amiga. If this makes you uncomfortable, please don't download this game!

This test version includes the game tutorial and a single level.

Please report issues using github issues here, or on Facebook or on EAB.
