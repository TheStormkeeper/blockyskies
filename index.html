BLOCKY SKIES AMIGA 2016

Download test version 3 of BLOCKY SKIES, a new game for classic Amiga.

This version is designed to run on any PAL 68k Amiga with 512kb of ram.

We have had reports of people succesfully running this version on Amiga 500, Amiga 1200 and Amiga 4000 hardware. It could still include bugs that might have unpredictable results.

This test version includes the game tutorial and a single playable level.

Please report issues using either github issues here, Facebook or EAB.
