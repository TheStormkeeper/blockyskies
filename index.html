BLOCKY SKIES AMIGA 2016

Download test version 4 of BLOCKY SKIES, a new game for classic Amiga.

Boot disk runs on any PAL 68k Amiga with 512kb of ram. Hard drive install requires 1mb chip ram. We have had reports of people succesfully running test version 3 on Amiga 500, Amiga 1200 and Amiga 4000 hardware.

This test version includes the game tutorial and a single playable level.

Please report issues using either github issues here, Facebook or EAB.
