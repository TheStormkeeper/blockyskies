BLOCKY SKIES AMIGA 2016

Download test version 7 of BLOCKY SKIES, a new game for classic Amiga.

Requirements:

  • PAL 68k Amiga
  • Boot disk version - 512kb chip ram
  • Hard drive install - 1mb chip ram
We have had reports of people succesfully running BLOCKY SKIES on Amiga 500, Amiga 1200, Amiga 4000 and CD32 hardware.

Test versions include a game tutorial and a single playable level.
