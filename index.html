BLOCKY SKIES AMIGA 2016

Download test version 1 of BLOCKY SKIES, a new game for classic Amiga.

This version is designed to run on any PAL 68k Amiga with 512kb of ram.

PLEASE NOTE: This code has not yet been tested on most real Amiga configurations. It could include bugs that might have unpredictable results.

If you are not comfortable testing unproven software on your precious Amiga, please do not download this game!!

This test version includes the game tutorial and a single playable level.

Please report issues using either github issues here, Facebook or EAB.
